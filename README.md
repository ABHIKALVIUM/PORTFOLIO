## ✨ System Architecture & UI/UX Features

This portfolio is built with a custom design system and highly optimized, zero-library animations utilizing `requestAnimationFrame` and `IntersectionObserver`.

- **Particle Canvas Engine:** A full-viewport vanilla JS background with interacting nodes and mouse-proximity physics.
- **Cinematic Dark Mode:** Deep `#0a0a0f` base layering with `#00d4ff` (electric cyan) and `#7c3aed` (deep purple) gradient accents.
- **Advanced Micro-Interactions:** - CSS `@keyframes` glitch typography effects.
  - 3D perspective tilt calculations on project cards based on mouse coordinates.
  - Custom dynamic cursor with lerped trailing lag.
- **Glassmorphism UI:** Blurred backdrop filters and conic-gradient glowing borders.
- **Optimized Performance:** Fully responsive, semantic HTML5, clean CSS architecture, and strict adherence to `prefers-reduced-motion` accessibility standards.

### 1. PrimeTradeAI
A production-grade monorepo separating a standalone REST API from a reactive React workspace. 
- **Key Features:** Implementations of RBAC, Redis caching (Cache-Aside & Write-Through), compound MongoDB indexing, and full Docker Compose containerization.
- **Tech:** React 19, Node.js, Express, MongoDB, Redis, Docker, JWT.
- [🔗 GitHub Repository](https://github.com/ABHIKALVIUM/PRIMETRADEAI) | [🌐 Live Demo](https://primetradeai-6vbl.vercel.app/)

### 2. KoinX Crypto Reconciliation Engine
A crypto transaction reconciliation engine utilizing advanced algorithms to match user and exchange data.
- **Key Features:** O(N log N) Two-Pointer/Sliding Window algorithm, streaming CSV ingestion, Zod validation, Recharts dashboard, and full Docker containerization.
- **Tech:** TypeScript, React 19, Node.js, Express, MongoDB, Recharts, Pino.
- [🔗 GitHub Repository](https://github.com/ABHIKALVIUM/KOINX-ASSIGNMENT)
